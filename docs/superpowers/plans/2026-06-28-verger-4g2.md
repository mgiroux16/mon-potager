> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Étendre le diagnostic IA pour qu'il analyse une photo et propose des pistes de traitement, ajouter une note de qualité de récolte par année sur la fiche arbre, et afficher une galerie photo datée de l'arbre.

**Architecture:** Trois sous-fonctionnalités indépendantes qui réutilisent toutes une infrastructure déjà en place (logs datés avec `photoUrls`, table `seasonNotes`, pipeline diagnostic Gemini) ; aucune nouvelle table Dexie, juste des champs optionnels ajoutés à des interfaces existantes.

**Tech Stack:** React 19, TypeScript, Dexie (IndexedDB), Vitest + Testing Library, API REST Gemini (`gemini-2.5-flash`, multimodal).

```typescript
/**
 * Extrait { data, mimeType } d'un data URL (ex: produit par compressImage pour photoUrls).
 * Renvoie null si la chaine ne suit pas le format `data:<mime>;base64,<data>`.
 */
export function parseDataUrl(url: string): { data: string; mimeType: string } | null {
  const match = /^data:([^;]+);base64,(.+)$/.exec(url)
  if (!match) return null
  return { mimeType: match[1], data: match[2] }
}
```
